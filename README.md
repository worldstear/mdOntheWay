# mdOntheWay
这是一个小白从零开始学习使用markdown的过程记录。
## 一、学习markdown的理由或初衷
- 不喜欢word复杂的样式，喜欢md简洁的纯文本格式。
- 平时不断学习的过程中多少都是需要用笔记记录下知识点或是思路之内的；但全靠脑袋记住很显然不靠谱。
